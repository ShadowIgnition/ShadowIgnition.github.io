TOOLS SHOWCASE

Effect Node Finder Tool

Developed for Lightmare Studios - Unity C#

A handy way to search all existing cards for any given effect or type.

Impact: Helps developers organise, debug and find obsolete card effects across hundreds of cards while also giving them the ability to filter for card stats, subtypes or by string, greatly improving efficiency when debugging / refactoring code.

Deck Collection Relocator Tool

Developed for Lightmare Studios - Unity C#

A tool that automatically moves cards into their respective folders and assigns them to their faction decks.

Impact: Helps developers automate tedious & time expensive work by automatically assigning cards to their faction decks and moves them into their allocated folders, saving developers between 5-30 minutes per major update.

Card Data Generation Tool

Developed for Lightmare Studios - Unity C#

A tool that allows developers to import & export json files for use in data

Impact: Helps developers organize data for the game, allowing better tracking of currently implemented cards, cards that require implementing, as well as cards that might hold incorrect data values.
